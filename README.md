# Setup

1. Clone the repo using
```
https://github.com/Arjun-G-04/portfolio.git
```

2. Install the modules using
```
yarn install
```

3. Run the site using
```
yarn dev
```

4. Voila! The site is up and running at http://localhost:3000/
